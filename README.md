# LEIA ESTE TEXTO ANTES DE CONTINUAR

Este projeto em HTML tem como objetivo ajudar estudantes do ensino médio, oferecendo materiais variados para auxiliar nos estudos, além de servir como apoio para ex estudantes que desejam aprender e melhorar seus conhecimentos com o conteúdo do ensino médio.

O site ainda está em desenvolvimento, mas a proposta do projeto é criar um material completo contendo:

* Matérias de Humanas
* Matéria de Exatas
* Matéria de Linguagens

O HTML principal (**index.html**) foi desenvolvido por **Cauê Shishido** e funciona como o menu principal do site, permitindo que o usuário escolha sua área de estudo e navegue pelo conteúdo.

Já os HTMLs secundários (**index/nome-da-opcao.html**) foram desenvolvidos por **Cauê Shishido** **David Delgado**, com o objetivo de proporcionar uma experiência de aprendizado prática e acessível para os usuários.
